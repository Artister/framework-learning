## About


> 前提： 需要本地安装nodejs, npm, yarn以及python2。具体安装请参考相应官网。
> 执行yarn run bootstrap安装mango命令之后, 便可在命令行中执行`mango create <app-name>`创建一个用于开发高级页面或标准页面资产的工程。

## Commands

##### yarn run bootstrap
安装mango命令
